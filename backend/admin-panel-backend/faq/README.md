# Trending Questions / FAQ Module

Auto-discover najczęściej zadawane pytania przez użytkowników chatbota.

## Files:
- `filters.py` - Pre-filtering junk messages
- `intent.py` - Intent detection (keyword-based)
- `gaps.py` - Gap detection (heuristic)
- `threshold.py` - Smart threshold algorithm
- `clustering.py` - Claude Haiku integration
- `analyzer.py` - Main Lambda handler
